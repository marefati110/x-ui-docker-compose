# x-ui-docker-compose


this repo is try to make easy to use x-ui with docker-compose

1) install docker

```bash
curl -fsSL https://get.docker.com | sh
```

2) clone repo

```bash
git clone https://github.com/marefati110/x-ui-docker-compose.git
``` 

3) run

```bash
docker compose up -d
```
